AudioView 모바일 앱 프론트엔드 디버깅 체크리스트
이 체크리스트는 AudioView Flutter 앱의 주요 기능과 UI 요소를 점검하기 위해 작성되었습니다.

1. 🔐 인증 및 세션 관리 (Auth Flow)
 로그인 테스트
 올바른 ID/PW 입력 시 로그인 성공 여부 (AuthProvider 상태 변경 확인)
 잘못된 정보 입력 시 에러 메시지 표시 여부
 로그인 버튼 중복 터치 방지 (로딩 인디케이터 표시)
 세션 유지
 앱 재시작 후 로그인 상태 유지 여부 (SharedPreferences 동작 확인)
 로그아웃
 로그아웃 시 홈 화면 또는 로그인 화면으로 적절히 이동하는지
 MY 탭 등 보호된 라우트 접근 시 로그인 유도
2. 🏠 홈 화면 (Home Screen)
 레이아웃 및 스크롤
 CustomScrollView 스크롤이 부드러운지 (Jank 확인)
 "새로 올라온 영화", "실시간 인기", "TOP 10" 섹션 간 간격 적절성
 카테고리 칩 (Chips)
 가로 스크롤 동작 확인
 칩 선택 시 시각적 피드백 (색상 변경 등)
 이미지 로딩
 영화 포스터(network image) 로딩 속도 및 캐싱
 로딩 실패 시 플레이스홀더(Fallback) 이미지 표시 여부
3. 🔍 검색 및 탐색 (Search)
 검색 기능
 검색어 입력 시 실시간 필터링 동작 여부
 검색 결과가 0건일 때 "결과 없음" 화면 표시
 키보드 Done 또는 Search 키 동작 처리
 필터 및 태그
 추천 검색어 태그 터치 시 검색창 입력 동작
 카테고리별 필터링 정확도
4. 👤 마이 페이지 및 보관함 (Downloads/My)
 접근 권한
 비로그인 상태에서 접근 시 "로그인이 필요합니다" 안내 표시
 데이터 표시
 시청 기록 목록 렌더링 정확성
 프로필 정보(이름, 등급 등) 표시 확인
5. ⚙️ 설정 및 접근성 (Settings)
 접근성 기능 (Accessibility)
 화면해설(AD) / 자막(CC) ON/OFF 토글 동작
 TalkBack / VoiceOver 지원 여부 (시각장애인용 앱이므로 필수)
 모든 이미지 버튼에 semanticLabel 적용되어 있는지
 포커스 순서가 논리적인지
 환경 설정
 "3G/LTE 사용 설정" 변경 시 상태 저장
 밝기 조절 오버레이(CustomHeader) 동작 확인
6. 📱 UI/UX 세부 사항
 네비게이션
 하단 탭(BottomNavigationBar) 전환 시 상태 유지 (IndexedStack 사용 여부)
 Android 뒤로가기 버튼(Back Button) 동작 (앱 종료 전 확인 모달 등)
 반응형 대응
 다양한 화면 크기(작은 폰 vs 대화면 폰)에서 레이아웃 깨짐 확인
 텍스트 오버플로우(Overflow) 처리 (긴 제목이 짤리는지, 줄바꿈 되는지)
 테마
 전체적인 다크 테마 색상 일관성 (scaffoldBackgroundColor, cardColor)
 텍스트 가독성 (배경색과의 명도 대비)
7. 🐛 에러 핸들링
 네트워크 에러
 오프라인 상태에서 앱 실행 시 안내 화면
 API 타임아웃 처리 (Mock 데이터 사용 시엔 로직만 점검)
